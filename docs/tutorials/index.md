## 시작하기 전에

Colab에서 실행하면 별도 설치가 필요 없다. 로컬에서 실행하려면 먼저 설치한다.

```python
uv add dartlab
```

데이터는 종목코드를 입력하면 GitHub Releases에서 자동으로 다운로드된다.
